SMART CAMPUS UPB - IoT

Plataforma IoT Inteligente para la Gestión Ambiental Comunitaria

Esta plataforma IoT ha sido diseñada para monitorear, analizar y gestionar variables ambientales en tiempo real, a partir de sensores distribuidos en un territorio definido. Utiliza el framework FIWARE como base operativa, lo cual garantiza interoperabilidad, escalabilidad y facilidad de integración con nuevas funcionalidades. Gracias a su arquitectura modular basada en microservicios, permite a las comunidades no solo visualizar datos históricos y actuales, sino también anticiparse a eventos críticos mediante el uso de herramientas avanzadas de análisis y predicción propias de la Cuarta Revolución Industrial.

La plataforma opera sobre el framework abierto FIWARE, reconocido internacionalmente por su capacidad de integración de datos contextuales en sistemas inteligentes. Esta base tecnológica proporciona los componentes necesarios para gestionar flujos de datos en tiempo real, normalizarlos y utilizarlos como insumo para tableros de control, alertas automatizadas y algoritmos de predicción.

Sistema comunitario y colaborativo

Lo construimos todos juntos... puedes participar!!!

Se trata de un sistema desarrollado para ser gestionado directamente por la comunidad. Esto asegura que las decisiones que se tomen estén fundamentadas en información precisa, generada localmente. La comunidad administra, mantiene y evoluciona la plataforma, utilizando tecnologías modernas para vigilar el entorno y actuar frente a condiciones adversas o cambios inesperados.

Plataforma inteligente, segura y adaptable

IaC, Contendedores, microservicios, Sensores

La plataforma cuenta con un módulo de monitoreo ambiental, gestión de recursos y administración del sistema, todo accesible mediante interfaces visuales intuitivas. Su estructura de microservicios permite incorporar nuevas funcionalidades según las necesidades emergentes, utilizando prácticas de desarrollo como Infraestructura como Código (IaC) y fomentando la contribución colaborativa de usuarios y desarrolladores.
